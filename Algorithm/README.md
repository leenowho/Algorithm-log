#  알고리즘 문제풀이 저장소

> 알고리즘 문제를 해결하며 학습한 내용을 폴더별로 정리한 저장소입니다.  

---

##  폴더 구조 

| 폴더명 | 설명 |
|--------|------|
| `BOJ_1244_250413` | 백준 1244번 문제 풀이 (2025년 4월 13일 풀이) |
| `SWEA_1789_250412` | SWEA 1789번 문제 풀이 (2025년 4월 12일 풀이) |

---

---

##  파일 구조

각 폴더 내부에는 아래와 같은 파일이 포함됩니다:

| 파일명 | 설명 |
|--------|------|
| `input.txt` | 테스트 케이스 |

| `SOL.py(성공)` | 파이썬 풀이 코드 + 접근 방식 주석 |
